# 크립토 선물 레짐 분류기 — 5상태 + OI 리드-래그 검증

목적은 분류 정확도가 아니라 **전환 조기 감지**다. 산출물 두 개:

1. **현재 상태 분류기** — 매 15m 캔들(번인 후)에 `{regime, confidence∈[0,1], timestamp}` 방출.
   5상태 `TREND_UP / TREND_DOWN / SQUEEZE / RANGE / CHOP`, 플레이북(`playbook.py`)으로 전략
   on/off 연결. 예측·목표가·시그널 없음.
2. **OI 융합의 조기성 검증** — OI 융합 라벨(F)이 베이스라인을 몇 캔들 선행하는가.
   레짐엔 그라운드 트루스가 없으므로 정확도는 지표가 아님. **핵심 숫자 = OI 한계기여**
   (= lead(F vs B1) − lead(B2 vs B1); B2 = 연료 축 뺀 동일 분류기).

설계 전문과 적대 검증(에이전트 6, 21개 급소) 반영 내역: `NOTES.md` + `~/.claude/plans/task-crypto-futures-warm-peacock.md`.

## 핵심 결과 (BTCUSDT 15m, 2022-01→2026-07, 154k 캔들)

| 지표 | 값 |
|---|---|
| OI 한계기여 (1차 엔드포인트) | enter-TREND 풀링 **+1캔들** [월클러스터 CI 1, 2] · UP +2 [1,2] · DOWN +1 [0,2] |
| **★ 순환 시간이동 널 (200회)** | 풀링: 널 **+1.0 [1,1], p=1.0** — 관측 +1과 **구별 불가**. 클래스별: UP 관측 +2 vs 널 +1, **p=0.144(ns)** · DOWN +1 = 널, p=1.0 |
| OI 1스냅샷(5분) 지연 강건성 | +1 [1,2] (동일 — 널 해석과 정합) |
| 원시 F vs B1 (참고용) | **공동 검출 에피소드 한정**(매칭률 0.42~0.47): 중앙값 −4(UP)/−6(DOWN), 원시 ADX보다 늦게 진입. 4개 era 전부 음(일관) |
| churn (3-슈퍼상태, flips/100봉) | B1 4.83 · **F 3.25** · B2 2.19 — F는 늦는 대신 훨씬 안정 |
| F 오경보율 | 캔들 기준 0.07/100봉 · **에피소드 기준 12.2%**(F 에피소드 1,649 중 202가 B1 무대응) — 두 분모 다 봐야 정직 |
| hysteresis | `confirm=1`: churn −26%(3.25→2.41)·플랩 0.23→0.06에 **리드 희생 0** — 공짜 안정화 |
| 무튜닝 복제 | ETH/SOL/XRP 셋 다 한계기여 +1 [1,2]·churn 서열·점유율 구조 동일 재현 |

**정직한 결론 (널 보정):** 관측된 +1캔들 조기화는 시간이동(정렬 파괴) OI로도 그대로 재현된다
(p=1.0). 통계 의미론 주의(외부감사 반영): p=1.0은 "우월 타이밍의 **증거 없음**"이지 동등성의
증명이 아니다(동등성 마진 검정은 미실시) — 실무 판단엔 충분하나 표현은 이렇게 읽어야 한다. 사분면의 방향 정보는 ΔOI가 아니라 **Δprice 레그**에서 나오므로, 이 +1은 "제3축이
실제 가격 방향으로 투표한다"는 구조 효과이지 **OI 정렬 정보가 아니다**. 즉 — 이 설정에서
**OI의 타이밍 알파는 검출되지 않았다.** 이 널 검정이 없었다면 "+1캔들 OI 엣지"(CI가 0을
제외!)를 오보로 출하했을 것이다 — 월클러스터 CI만으로는 부족했다는 것이 이 리서치의 핵심
방법론적 교훈. collapse 각도(2차)는 precision 0.982 vs 널 0.974(p=0.005)로 유의하지만
효과량이 미미하고 recall은 ns(p=0.114) — 약한 신호. (전 수치는 적대적 코드리뷰 후 교정된
기계로 재산출한 최종값 — 수정 전후 결론 불변이 그 자체로 강건성 증거.)

분류기 자체는 결과와 무관하게 출하: 5상태 전부 활성(점유율 9.6~48%), 체류 중앙값 7~13봉,
B1 대비 churn −33%·오경보 근절 — "늦지만 깨끗한" 레짐 게이트로서의 가치는 리드-래그와 별개.

**R2 추가 검증 (2026-07-05):**
- **사후 성과 감사**(`--stage posthoc`): 변동성 서열 완벽(24h |폭|: 수렴 0.88% < 박스 1.21% <
  추세 1.4%대 < 난장 1.59%) — 상태 의미는 실재. 방향 지속성은 없음(추세 라벨 후 미세 되돌림)
  — **라벨은 시장의 '성격' 기술이지 방향 예측이 아니다.**
- **무료 파생 신호 3종 전패**: 테이커 흐름·톱트레이더 LSR·DVOL(내재변동성) 전부 한계기여
  0.0, 시간이동 널 p=1.0 (사전등록 인코딩, `alt_axes.py`). OI 포함 네 신호가 enter-TREND
  타이밍에 무기여 — 남은 후보는 청산 이벤트(유료) 또는 타깃 변경(변동성 팽창 타이밍).
- `current.py`에 **멀티TF(15m/1h/4h) 합의 뷰**, Pine v1.1(차트 위 한글 테이블·범례·전환 마커).

## 매매 적용 프로토콜 (선물 레버리지 데이트레이딩, R:R 게이트)

검증이 말해주는 사용법: **이 지표는 진입 시그널이 아니라 "어떤 전략을 켜고, 얼마나 걸까"의
게이트**다. 15m 라벨엔 방향 예측력이 없고(사후 감사), 143k 백테스트 생존 통계도 방향
결정은 1H/4H에서 하라고 말한다. 위계:

1. **방향·전략 클래스 = 4H(주)·1H(보조) 레짐** — `python current.py`의 MTF 판정 또는
   지표의 "4H 방향" 행. 4H가 추세면 그 방향만, 비추세면 방향 베팅 금지.
2. **진입 타이밍·회피 = 15m 레짐** — 4H 방향과 15m 상태 조합으로 셋업 선택.
3. **사이즈/레버리지 = 확신 등급 + 전환경고** — 확신 '낮음' 또는 ⚠경고 시 절반, CHOP은 0.

| 4H \ 15m | 🟢/🔴 추세 | 🔵 수렴 | ⚪ 박스권 | 🟠 난장판 |
|---|---|---|---|---|
| **추세 (4H)** | 눌림목/되돌림 진입, R:R≥2, 구조 SL | 확장 '확인 후' 추세방향만 | 박스 이탈-리테스트만, R:R≥2 | 관망 |
| **비추세 (4H)** | 15m 단독추세 추격 금지 | 대기 (방향 없음) | 박스 역매매, R:R 1.5, 짧은 SL | 관망 |

- 청산 정책·트리거는 HT1 v2 확정안 유지 — 이 게이트는 그 앞단의 on/off 스위치.
- 돌파는 통계적 최약체 전략군(생존 19/196): 수렴 돌파는 반드시 확장 확인(또는 리테스트) 후.

## 정직성 규약 (요약)

- 룩어헤드 테스트는 **원시 클라인·metrics·펀딩 3테이블 전부**를 벽시계 t*에서 절단/오염하고
  파이프라인 전체를 재실행, t* 이전 출력의 정확 일치를 요구. 카나리(OI −1캔들 시프트)가
  **실제로 빨간불**이 되는 것까지 테스트함.
- 매칭은 에피소드 단위·H 파라미터 없음. 오경보율이 1급 지표. 매처 자가검증 3종
  (노이지 비선행 ~0 / 노이지 선행 k 복원 / 갭)을 실데이터 전에 통과.
- 모든 분위수는 과거 전용. 번인 35일 제외. 갭 인접 에피소드 통계 제외.
- 파라미터는 실데이터 접촉 전 동결. 민감도 그리드(k×데드존, d×겹침)로 knife-edge 여부 공개.

- 30일 롤링 분위수 창보다 오래 지속되는 레짐은 서서히 재정규화됨(적응형 임계값의 본질).
- 리드-래그는 B1과 F가 공동 검출한 에피소드(매칭률 ~0.44)에 한정 — B1 단독 에피소드 대부분은
  F가 아예 추세로 보지 않은 단명 구간(오경보일 수도, 미탐일 수도 — 그라운드 트루스 없음).
- **외부 감사(Codex/GPT-5.5, 2026-07-05, `results/codex_review.log`) 반영 한계**: 월 클러스터
  부트스트랩은 리샘플마다 에피소드 재매칭을 하지 않아 CI가 과소일 수 있음(재매칭 블록
  부트스트랩은 향후 과제). 순환 시프트 널은 비정상성 하에서 완전한 무작위화가 아니며
  2~90d 범위 제한 — 느린 신호의 위음성 가능성 잔존. 펀딩 스냅(≤60초)은 미세한 조기 가용
  가정. 룩어헤드 카나리는 OI 경로만 커버(클라인/펀딩 경로 카나리는 향후). 감사 총평:
  "코어 분류는 실질적으로 인과적, 결론은 탐색적 수준" — 이 리서치의 자기평가와 일치.
